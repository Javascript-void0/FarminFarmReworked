## Reference

 > Numeral

Numbers on the server are represented differently than conventional 'K', 'M', 'B', 'T', etc. Instead of these, the server represents thousands, millions, billions, etc. with lowercase alphabet letters. Starting with 'a', representing thousands, then to 'b', representing millions.

 # Ex: 12.2a = ~12,200 (rounded)
 # Ex: 109.3b = ~109,300,000
 # Ex: 4.8b = ~4,800,000,000

Knowing exact numbers isn't really important, but you can often hover over these abbreviated numbers in chat to reveal its exact, un-rounded value. Skript's variables allow the numbers to go up to 'f'.

 > Chat-UI

Umoria uses Minecraft chat as UI. I opted to try this because I can show more information through chat, and the original Umoria is a game played by displaying text in a terminal window.


Because of this design choice, Minecraft's chat is designated to be used for chat-ui and other gameplay features. Chatting is still avaibale, but messages will not show up in-game, only in #umoria in Discord. Click here for more info

Overlay :


In-game chat (Minecraft chat) is disabled because it might interfere with the chat-ui. But, chat is still avaiable in Discord in #umoria. If you really want to view chat, I would recommend either having Discord open on the side, or using Discord's overlay feature.


To use the overlay, go into settings and add Minecraft to Activity Status. Make sure OVERLAY: ON in Activity Status for Minecraft. Then go to Game Overlay menu and make sure it is also on. Open minecraft and press Shift + ` (or whatever keybind you have it set to)
